# 关系选择器

用于根据元素之间的层级（父子、祖孙）或位置（兄弟）关系进行精准选择。

## 并集选择器

使用逗号 `,` 分隔多个选择器，同时选中这些选择器对应的**所有**元素。这也称为“群组选择器”。

- **语法**：`选择器1, 选择器2 { 属性: 值; }`
- **用途**：用于给不同的元素设置相同的公共样式，减少代码冗余。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    /* 让 h4 和 p 共享相同的颜色 */<br>
    h4, p { color: #2196f3; }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;h4&gt;我是标题&lt;/h4&gt;<br>
    &lt;p&gt;我是段落&lt;/p&gt;<br>
    &lt;div&gt;我是无关的 div&lt;/div&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <h4 style="color: #2196f3; margin: 0 0 10px 0;">我是标题</h4>
    <p style="color: #2196f3; margin: 0 0 10px 0;">我是段落</p>
    <div style="color: #000;">我是无关的 div</div>
  </div>
</div>

## 交集选择器

将两个选择器直接连接（无空格），选中**同时满足**这两个条件的元素。

- **语法**：`标签名.类名 { ... }` 或 `标签名#ID { ... }`
- **注意**：标签选择器必须写在最前面（例如 `div.active` 是合法的，而 `.active.div` 虽在某些预处理器可行，但原生 CSS 推荐标准写法）。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    /* 仅选中 class 为 active 的 div 元素 */<br>
    div.active { <br>
    &nbsp;&nbsp;background-color: #e8f5e9;<br>
    &nbsp;&nbsp;border: 1px solid #4caf50;<br>
    }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;div class="active"&gt;生效：是 div 且有 active&lt;/div&gt;<br>
    &lt;p class="active"&gt;无效：是 p 即使有 active&lt;/p&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <div style="background-color: #e8f5e9; border: 1px solid #4caf50; padding: 10px; margin-bottom: 10px;">生效：是 div 且有 active</div>
    <p style="background-color: transparent; border: 1px dashed #ccc; padding: 10px; margin: 0;">无效：是 p 即使有 active</p>
  </div>
</div>

## 后代选择器

使用**空格**分隔，选中某元素内部的**所有**指定后代元素。

- **语法**：`祖先 后代 { ... }`
- **范围**：包括子元素、孙元素、重孙元素等所有层级。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    /* 选中 .box 内部所有的 span */<br>
    .box span { color: red; font-weight: bold; }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;div class="box"&gt;<br>
    &nbsp;&nbsp;&lt;span&gt;子元素 span (生效)&lt;/span&gt;<br>
    &nbsp;&nbsp;&lt;p&gt;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&lt;span&gt;孙元素 span (也生效)&lt;/span&gt;<br>
    &nbsp;&nbsp;&lt;/p&gt;<br>
    &lt;/div&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <div style="border: 1px dashed #ccc; padding: 10px;">
      <span style="color: red; font-weight: bold;">子元素 span (生效)</span>
      <p style="margin: 10px 0 0 20px; border-left: 2px solid #eee; padding-left: 10px;">
        <span style="color: red; font-weight: bold;">孙元素 span (也生效)</span>
      </p>
    </div>
  </div>
</div>

## 子代选择器

使用大于号 `>` 分隔，仅选中某元素的**直接**子元素。

- **语法**：`父元素 > 子元素 { ... }`
- **范围**：不包含孙元素及更深层的后代。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    /* 仅选中 list 的直接 li 子元素 */<br>
    .list > li { border-bottom: 2px solid #2196f3; }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;ul class="list"&gt;<br>
    &nbsp;&nbsp;&lt;li&gt;第一级菜单 (生效)<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&lt;ul&gt;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;li&gt;第二级菜单 (无效，这是孙子)&lt;/li&gt;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&lt;/ul&gt;<br>
    &nbsp;&nbsp;&lt;/li&gt;<br>
    &lt;/ul&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <ul style="list-style: none; padding: 0; margin: 0;">
      <li style="border-bottom: 2px solid #2196f3; padding: 5px 0;">
        第一级菜单 (生效)
        <ul style="list-style: none; padding-left: 20px; margin-top: 5px;">
          <li style="border-bottom: 1px dashed #ccc; color: #999;">第二级菜单 (无效，这是孙子)</li>
        </ul>
      </li>
    </ul>
  </div>
</div>

## 相邻兄弟选择器

使用加号 `+` 分隔，选中**紧接**在某元素后的**第一个**同级元素。

- **语法**：`元素A + 元素B { ... }`
- **条件**：元素 B 必须紧跟在 元素 A 后面，且二者拥有同一个父元素。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    /* 选中紧跟在 h3 后的第一个 p */<br>
    h3 + p { color: red; }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;h3&gt;标题&lt;/h3&gt;<br>
    &lt;p&gt;第一个段落 (变红)&lt;/p&gt;<br>
    &lt;p&gt;第二个段落 (不变)&lt;/p&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <h3 style="margin: 0 0 10px 0;">标题</h3>
    <p style="color: red; margin: 0 0 10px 0;">第一个段落 (变红)</p>
    <p style="color: #000; margin: 0;">第二个段落 (不变)</p>
  </div>
</div>
