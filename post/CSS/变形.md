# 变形 (Transforms)

CSS3 的 `transform` 属性允许我们在 2D 或 3D 空间中对元素进行旋转、缩放、倾斜或平移。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
<strong>不脱离文档流</strong><br>
变形通常只影响元素的视觉表现，不会改变其在文档流中实际占据的空间（周围的元素不会因为这个元素变大了而被挤开）。

## 2D 变换函数

以下是四种最基础的 2D 变换函数。

- **平移 (`translate`)**：移动元素。
    - `translate(x, y)`：沿 X 轴和 Y 轴移动。
    - **技巧**：若使用百分比（如 `50%`），是相对于**自身尺寸**计算的。常用于绝对定位居中：`translate(-50%, -50%)`。
- **缩放 (`scale`)**：改变大小。
    - `scale(n)`：整体缩放 n 倍。`1` 为原始大小，`>1` 放大，`0-1` 缩小。
    - `scale(x, y)`：分别控制水平和垂直缩放。
- **旋转 (`rotate`)**：旋转元素。
    - `rotate(angle)`：正值为顺时针，负值为逆时针（单位 `deg`）。
    - 默认旋转中心是元素的中心点（可通过 `transform-origin` 修改）。
- **倾斜 (`skew`)**：使元素发生剪切畸变。
    - `skew(x-angle, y-angle)`：通常只设置 X 轴倾斜，产生平行四边形效果。

## 组合变换

`transform` 属性可以接受多个函数，使用空格分隔。

- **语法**：`transform: 函数1 函数2 ...;`
- **注意**：**执行顺序是从左到右**。先旋转再平移，与先平移再旋转，由于坐标系也会随之旋转，最终位置可能完全不同。
