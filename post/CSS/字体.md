# 字体

用于控制文本的排版样式，如倾斜、粗细、大小和字体系列。

## 字体样式

属性 `font-style` 用于定义文本的风格。

- **`normal`**：正常显示。
- **`italic`**：斜体。使用字体库中专门设计的“斜体字形”。
- **`oblique`**：倾斜。如果字体没有斜体变体，浏览器会通过算法强制将文字向右倾斜。

## 字体外观

通过粗细、大小和字体系列来定义文本的视觉表现。

- **粗细 (`font-weight`)**：
    - 关键字：`normal` (400), `bold` (700)。
    - 数值：`100` 到 `900`（需字体文件支持对应的粗细等级）。
- **大小 (`font-size`)**：
    - `px`：像素，绝对单位，精确。
    - `rem`：相对于根元素 (`html`) 的字体大小，推荐用于响应式布局。
- **系列 (`font-family`)**：
    - 指定字体名称列表，浏览器会按顺序尝试加载。
    - **最佳实践**：始终在最后加上通用字体族（如 `sans-serif` 或 `serif`）作为保底。

## 综合属性

`font` 属性可以将上述所有属性合并为一行书写。

- **语法顺序**（严格）：`font-style font-weight font-size/line-height font-family`
- **注意**：
    - `font-size` 和 `font-family` 是**必填项**，否则声明无效。
    - `line-height`（行高）必须紧跟在 `font-size` 后面，用 `/` 分隔。
