# 过渡 (Transitions)

过渡允许 CSS 属性值在一定时间区间内平滑地改变，而不是瞬间跳变。这通常用于增强用户交互体验（如按钮悬停效果）。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
<strong>触发机制</strong><br>
过渡不是一种自动播放的动画，它必须由状态变化触发，例如 <code>:hover</code>、<code>:focus</code>、<code>:active</code> 或通过 JavaScript 修改类名。
</div>

## 核心属性

- **`transition-property`**：指定应用过渡效果的 CSS 属性名称（如 `background-color`, `width`, `transform`）。
    - 建议：尽量避免使用 `all`，指定具体属性有助于提升性能。
- **`transition-duration`**：过渡持续的时间（单位 `s` 秒或 `ms` 毫秒）。
- **`transition-timing-function`**：速度曲线（时间函数），控制变化节奏。
    - `linear`：匀速。
    - `ease`：默认值，慢启动 -> 加速 -> 慢结束。
    - `ease-in`：慢启动。
    - `ease-out`：慢结束。
    - `steps(n)`：分 n 步跳变。
- **`transition-delay`**：延迟开始的时间。

## 简写语法

推荐使用 `transition` 简写属性一次性设置。

- **语法**：`transition: 属性 持续时间 速度曲线 延迟;`
- **解析规则**：浏览器会根据数值出现的顺序进行解析，**第一个时间值**总是被视为持续时间，**第二个时间值**（如果有）被视为延迟时间。

<div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
&lt;!-- 源码：多属性过渡 --&gt;<br>
/* 写法 1：分别指定（中间用逗号分隔） */<br>
transition: width 0.5s ease, background 0.3s linear;<br>
<br>
/* 写法 2：全部属性（简单但可能有性能损耗） */<br>
transition: all 0.5s ease-in-out;
</div>

## 演示：综合应用

结合 `transform` 和 `transition` 制作一个交互卡片。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
<div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
&lt;!-- CSS 源码 --&gt;<br>
.card {<br>
&nbsp;&nbsp;width: 100px; height: 100px;<br>
&nbsp;&nbsp;background: #2196f3;<br>
&nbsp;&nbsp;/* 宽度变化慢，旋转变化快 */<br>
&nbsp;&nbsp;transition: width 1s, transform 0.5s;<br>
}<br>
<br>
.card:hover {<br>
&nbsp;&nbsp;width: 200px;<br>
&nbsp;&nbsp;transform: rotate(10deg);<br>
}
</div>
<div style="padding: 20px; background-color: #fff; height: 150px;">
<div style="width: 100px; height: 100px; background: #2196f3; color: white; display: flex; align-items: center; justify-content: center; font-size: 12px; transition: width 1s, transform 0.5s; cursor: pointer; border-radius: 4px; box-shadow: 2px 2px 5px rgba(0,0,0,0.2);" onmouseover="this.style.width='200px'; this.style.transform='rotate(10deg)'" onmouseout="this.style.width='100px'; this.style.transform='none'">
Hover Me
</div>
</div>
