# CSS 引入

CSS（层叠样式表）样式应用到 HTML 文档中主要有三种方式，每种方式的作用范围、维护成本以及优先级均不同。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
  <strong>核心原则：关注点分离 (Separation of Concerns)</strong><br>
  在现代 Web 开发中，推荐尽可能使用<strong>外部样式表</strong>。这能让 HTML 专注结构，CSS 专注表现，JS 专注行为，极大降低维护难度。
</div>

## 行内样式

直接在 HTML 标签的 `style` 属性中定义 CSS 样式。

- **定义**：将 CSS 规则写在元素的 `style="..."` 属性中。
- **优先级**：**最高**。它会覆盖外部样式表和内部样式表中的同名规则。
- **缺点**：导致结构与表现混合，代码冗余，无法复用，极难维护。仅用于测试或通过 JS 动态改变样式。

  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">

    &lt;div style="color: #f44336; font-weight: bold; border: 1px solid #f44336; padding: 10px;"&gt;<br>
    &nbsp;&nbsp;这是一个行内样式的例子<br>
    &lt;/div&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <div style="color: #f44336; font-weight: bold; border: 1px solid #f44336; padding: 10px;">
      这是一个行内样式的例子
    </div>
  </div>
</div>

## 内部样式表

在 HTML 文档的 `<head>` 标签中使用 `<style>` 标签定义 CSS 样式。

- **定义**：样式代码包裹在 `<style>` 标签中。
- **作用域**：仅对**当前页面**有效。
- **适用场景**：单页面应用（SPA）的组件样式，或者邮件模板（Email HTML）等无法加载外部资源的场景。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- 源码结构 --&gt;<br>
    &lt;head&gt;<br>
    &nbsp;&nbsp;&lt;style&gt;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;/* 定义一个类选择器 */<br>
    &nbsp;&nbsp;&nbsp;&nbsp;.box-internal {<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;background-color: #e8f5e9;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;color: #2e7d32;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;padding: 10px;<br>
    &nbsp;&nbsp;&nbsp;&nbsp;}<br>
    &nbsp;&nbsp;&lt;/style&gt;<br>
    &lt;/head&gt;<br>
    &lt;body&gt;<br>
    &nbsp;&nbsp;&lt;div class="box-internal"&gt;通过内部样式表控制的元素&lt;/div&gt;<br>
    &lt;/body&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <div style="background-color: #e8f5e9; color: #2e7d32; padding: 10px;">
      通过内部样式表控制的元素
    </div>
  </div>
</div>

## 外部样式表

将 CSS 代码编写在独立的 `.css` 文件中，通过 `<link>` 标签引入 HTML 文档。

- **定义**：使用 `<link rel="stylesheet" href="...">` 引入文件。
- **优势**：
    - **复用性**：一个样式表可控制整个网站成千上万个页面。
    - **缓存**：浏览器会缓存 CSS 文件，加快页面二次加载速度。
    - **维护性**：完全实现结构与表现分离，是开发中的标准做法。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- 源码：index.html --&gt;<br>
    &lt;head&gt;<br>
    &nbsp;&nbsp;&lt;!-- 引入外部文件 --&gt;<br>
    &nbsp;&nbsp;&lt;link rel="stylesheet" href="style.css"&gt;<br>
    &lt;/head&gt;<br>
    &lt;body&gt;<br>
    &nbsp;&nbsp;&lt;button class="btn-primary"&gt;外部样式按钮&lt;/button&gt;<br>
    &lt;/body&gt;<br>
    <br>
    &lt;!-- 源码：style.css --&gt;<br>
    .btn-primary {<br>
    &nbsp;&nbsp;background-color: #2196f3;<br>
    &nbsp;&nbsp;color: white;<br>
    &nbsp;&nbsp;border: none;<br>
    &nbsp;&nbsp;padding: 8px 16px;<br>
    &nbsp;&nbsp;border-radius: 4px;<br>
    }
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <button style="background-color: #2196f3; color: white; border: none; padding: 8px 16px; border-radius: 4px; cursor: pointer;">外部样式按钮</button>
  </div>
</div>
