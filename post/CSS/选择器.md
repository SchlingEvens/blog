# 选择器

选择器用于指定 CSS 样式作用的 HTML 元素，是 CSS 规则的第一部分。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
  <strong>权重概念</strong><br>
  不同选择器的优先级不同：<strong>ID 选择器 > 类选择器 > 标记选择器</strong>。<br>
  如果样式发生冲突，浏览器会应用权重更高的规则。
</div>

## 标记选择器

使用 HTML 标签名称作为选择器，选中页面中**所有**该类标签。

- **定义**：直接写标签名（如 `p`, `div`, `h1`）。
- **作用**：设置网页的默认样式（如统一所有段落的行高）。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    p {<br>
    &nbsp;&nbsp;color: #555;<br>
    &nbsp;&nbsp;line-height: 1.5;<br>
    }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;p&gt;第一段文字...&lt;/p&gt;<br>
    &lt;p&gt;第二段文字...&lt;/p&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <p style="color: #555; line-height: 1.5; margin-bottom: 10px;">第一段文字：所有 p 标签都会应用此样式。</p>
    <p style="color: #555; line-height: 1.5; margin: 0;">第二段文字：无需添加类名，自动生效。</p>
  </div>
</div>

## ID 选择器

使用元素的 `id` 属性值作为选择器。

- **语法**：以 `#` 开头（如 `#header`）。
- **规范**：HTML 页面中 ID 必须是**唯一**的。因此，ID 选择器用于选中页面中特定的、唯一的元素（如页眉、侧边栏容器）。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    #submit-btn {<br>
    &nbsp;&nbsp;background-color: #000;<br>
    &nbsp;&nbsp;color: #fff;<br>
    &nbsp;&nbsp;padding: 10px 20px;<br>
    }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;button id="submit-btn"&gt;提交订单&lt;/button&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <button style="background-color: #000; color: #fff; padding: 10px 20px; border: none; cursor: pointer;">提交订单</button>
  </div>
</div>

## 类选择器

使用元素的 `class` 属性值作为选择器。

- **语法**：以 `.` 开头（如 `.highlight`）。
- **规范**：
    - **复用性**：同一个类名可以应用于多个不同的元素。
    - **组合性**：一个 HTML 元素可以同时拥有多个类名（如 `class="box highlight"`）。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- CSS 源码 --&gt;<br>
    /* 定义基础卡片样式 */<br>
    .card {<br>
    &nbsp;&nbsp;border: 1px solid #ddd;<br>
    &nbsp;&nbsp;padding: 10px;<br>
    &nbsp;&nbsp;display: inline-block;<br>
    }<br>
    /* 定义高亮样式 */<br>
    .highlight {<br>
    &nbsp;&nbsp;background-color: #fff3e0;<br>
    &nbsp;&nbsp;color: #e65100;<br>
    &nbsp;&nbsp;border-color: #ff9800;<br>
    }<br>
    <br>
    &lt;!-- HTML 源码 --&gt;<br>
    &lt;div class="card"&gt;普通卡片&lt;/div&gt;<br>
    &lt;div class="card highlight"&gt;高亮卡片&lt;/div&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <div style="border: 1px solid #ddd; padding: 10px; display: inline-block; margin-right: 10px;">普通卡片</div>
    <div style="border: 1px solid #ff9800; padding: 10px; display: inline-block; background-color: #fff3e0; color: #e65100;">高亮卡片</div>
  </div>
</div>

## 通配符选择器

选中页面中的**所有**元素。

- **语法**：使用 `*` 符号。
- **用途**：常用于 CSS Reset（样式重置），消除浏览器默认的内外边距。
- **注意**：由于涉及遍历页面所有节点，在大型项目中需谨慎使用，现代开发常用 `normalize.css` 代替简单的通配符重置。
