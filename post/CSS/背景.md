# 背景

用于设置元素的背景样式，包括颜色、图片、平铺方式以及现代 CSS 支持的渐变效果。

## 背景属性

使用 `background-*` 系列属性控制背景的各个方面。

- **`background-color`**：背景颜色。
- **`background-image`**：背景图片，格式为 `url('路径')`。
- **`background-repeat`**：平铺方式。
    - `repeat`：默认，水平垂直都平铺。
    - `no-repeat`：不平铺，只显示一次。
- **`background-position`**：图片位置。可以使用关键字（`center`, `top`）或数值（`50% 50%`, `10px 20px`）。
- **`background-size`**：图片尺寸（**核心属性**）。
    - `cover`：等比例缩放图片以**填满**容器（可能裁剪图片）。
    - `contain`：等比例缩放图片以**完整显示**在容器内（可能留白）。
- **`background-attachment`**：背景附着。
    - `scroll`：背景随页面滚动（默认）。
    - `fixed`：背景固定在视口，不随页面滚动。

## 渐变属性

CSS3 允许使用渐变函数来代替纯色图片，具有更高的性能和灵活性。

- **线性渐变 (`linear-gradient`)**：
    - 语法：`linear-gradient(方向, 颜色1, 颜色2...)`
    - 方向：`to right` (向右), `to bottom right` (向右下), `45deg` (45度)。
- **径向渐变 (`radial-gradient`)**：
    - 语法：`radial-gradient(形状, 颜色1, 颜色2...)`
    - 形状：`circle` (圆形), `ellipse` (椭圆，默认)。

## 简写属性

可以使用 `background` 属性一次性设置所有背景样式。

- **推荐顺序**：颜色 图片 平铺 附着 位置 / 尺寸
- **示例**：`background: #fff url('img.png') no-repeat center / cover;`
