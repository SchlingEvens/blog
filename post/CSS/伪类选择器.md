# 伪类选择器

伪类用于定义元素的特殊状态（如被点击、被悬停）或特定的结构位置（如第一个子元素）。

## 动态伪类

用于描述用户与元素交互时的状态。

- **状态列表**：
    - `:link`：未访问的链接（仅适用于 `<a>`）。
    - `:visited`：已访问的链接（出于隐私安全，浏览器严格限制此状态下能修改的样式）。
    - `:hover`：鼠标悬停在元素上的状态。
    - `:active`：鼠标按下但未弹起的状态。
- **LVHA 原则**：定义 CSS 时必须严格遵循 **L**ink -> **V**isited -> **H**over -> **A**ctive 的顺序，否则后续样式会被前面的覆盖。

## 结构伪类

用于根据元素在父元素中的位置进行选择。

- **`:first-child`**：选中父元素的第一个子元素。
- **`:last-child`**：选中父元素的最后一个子元素。
- **`:nth-child(n)`**：选中父元素的第 n 个子元素。
    - `nth-child(2)`：第 2 个。
    - `nth-child(odd)`：所有奇数项。
    - `nth-child(even)`：所有偶数项。

## 关系伪类 :has()

`:has()` 被称为 CSS 的“父选择器”，它允许我们根据**子元素**的状态来改变**父元素**的样式。

- **定义**：`父元素:has(子元素)` —— 当父元素包含特定的子元素时生效。
- **用途**：可以根据卡片是否包含图片来改变卡片的背景色或布局。
