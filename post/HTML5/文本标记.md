# HTML 文本标记

文本标记用于定义文档的文本内容及其逻辑层级结构，是构建语义化网页信息的基础。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
  <strong>核心概念：语义化</strong><br>
  使用正确的标签包含内容（例如使用 h1 而不是加粗的 div），能让屏幕阅读器和搜索引擎更好地理解网页结构。
</div>

## 标题标签

HTML 提供了六个等级的标题标签，即 `<h1>` 到 `<h6>`。

- **层级定义**：`<h1>` 定义最高等级（页面主旨），`<h6>` 定义最低等级。
- **SEO 权重**：`<h1>` 权重最高，通常一个页面仅建议出现一次。
- **视觉误区**：**禁止**纯粹为了放大字体或加粗而使用标题标签，样式控制应交给 CSS。
- **默认样式**：浏览器会自动为标题添加上下外边距 (`margin`)。

## 段落标签

`<p>` 标签用于定义文档中的文本段落。

- **块级元素**：`<p>` 是块级元素，独占一行，且默认具有上下间距。
- **DOM 解析规则**：**禁止**在 `<p>` 标签内部嵌套其他块级元素（如 `<div>`, `<h1>`, `<ul>` 等）。如果这样做，浏览器会自动强制闭合段落，导致 DOM 结构错乱。

<div style="background-color: #ffebee; border-left: 5px solid #f44336; padding: 15px; margin: 10px 0; color: #b71c1c;">
  <strong>错误示范：</strong><br>
  <code>&lt;p&gt;一段文字 &lt;div&gt;块级元素&lt;/div&gt;&lt;/p&gt;</code><br>
  浏览器会将其解析为：<code>&lt;p&gt;一段文字&lt;/p&gt;&lt;div&gt;块级元素&lt;/div&gt;&lt;p&gt;&lt;/p&gt;</code>
</div>

## 换行标签

`<br>` 标签用于插入一个简单的换行符。

- **空元素**：属于空元素，没有结束标签。
- **使用场景**：仅用于具有语义的换行（如地址、诗歌）。
- **反模式**：**严禁**使用连续的 `<br>` 来制造垂直间距。布局间距应始终使用 CSS 的 `margin` 或 `padding`。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- 源码 --&gt;<br>
    北京市海淀区&lt;br&gt;<br>
    中关村大街1号&lt;br&gt;<br>
    100000
  </div>
  <div style="padding: 20px; background-color: #fff; line-height: 1.5;">
    北京市海淀区<br>
    中关村大街1号<br>
    100000
  </div>
</div>

## 水平线标签

`<hr>` 标签在 HTML5 中被重新定义为段落级别的“主题转换”。

- **表现形式**：默认渲染为一条水平线，但语义上表示话题的变更。
- **样式化**：可以通过 CSS 修改其颜色、高度和边框样式。
