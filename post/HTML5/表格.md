# 列表标记笔记

列表用于将相关联的数据条目化展示，是网页排版和构建导航菜单的基础结构。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
  <strong>最佳实践：列表嵌套</strong><br>
  列表可以相互嵌套。例如，在 <code>&lt;li&gt;</code> 内部可以包含另一个 <code>&lt;ul&gt;</code>，这通常用于构建多级下拉菜单或目录树。
</div>

## 无序列表

无序列表用于展示项目之间没有顺序关系的数据集合。

- **标签**：`<ul>` (Unordered List) 定义容器，`<li>` (List Item) 定义列表项。
- **默认样式**：列表项前显示实心圆点（Bullet points）。
- **样式定制**：可以通过 CSS 的 `list-style-type` 属性将圆点修改为方块、空心圆或自定义图标。

## 有序列表

有序列表用于展示具有先后顺序或优先级的项目。

- **标签**：`<ol>` (Ordered List) 定义容器，`<li>` 定义列表项。
- **默认样式**：列表项前显示阿拉伯数字（1, 2, 3...）。
- **核心属性**：
    - `type`：指定序号类型（`1`=数字, `A`=大写字母, `a`=小写字母, `I`=大写罗马数字, `i`=小写罗马数字）。
    - `start`：指定列表的起始数值（例如从 5 开始）。

## 定义列表

定义列表用于展示术语及其描述，或“键-值”对数据。

- **标签结构**：
    - `<dl>` (Description List)：列表容器。
    - `<dt>` (Description Term)：定义术语或名字（Key）。
    - `<dd>` (Description Details)：定义描述或解释（Value）。
- **对应关系**：一个 `<dt>` 可以对应多个 `<dd>`，反之亦然。
- **默认样式**：`<dd>` 通常会在视觉上产生缩进效果。
