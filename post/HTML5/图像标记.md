# 图像标记

图像标记用于在网页结构中嵌入视觉媒体文件，是增强网页表现力的核心元素。

## 图像标签

HTML 使用 `<img>` 标签来定义图像。

- **空元素**：`<img>` 是空元素，只包含属性，没有闭合标签。
- **显示模式**：默认为行内块元素（Inline-block），这意味着它允许设置宽高，但不会独占一行。
- **底部间隙**：由于是为了配合文本基线对齐，图片底部默认会留有少量空白，通常可以通过 CSS `display: block` 或 `vertical-align: bottom` 消除。

## 核心属性详解

### src (Source)
指定图像文件的路径（URL），是必须属性。
- **相对路径**：如 `images/logo.png`（引用本站资源，推荐）。
- **绝对路径**：如 `https://www.example.com/logo.png`（引用外部资源）。

### alt (Alternative Text)
规定图像的替代文本，是必须属性。
- **加载失败**：当图片路径错误或网速过慢时，显示此文本。
- **无障碍访问**：屏幕阅读器会朗读此内容帮助视障用户理解。
- **SEO**：搜索引擎通过此属性索引图片内容。

### width / height
定义图像的显示尺寸（单位默认像素）。
- **防止抖动**：明确指定宽高可以让浏览器预留位置，防止图片加载过程中页面布局发生跳动（CLS - 累积布局偏移）。
- **性能建议**：应尽量加载与显示尺寸匹配的图片，避免加载过大图片后通过 HTML 强行缩小。
