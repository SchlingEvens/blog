# 超链接标记

超链接是万维网（WWW）的核心特性，它将分散的网页连接成网状结构，允许用户从一个页面跳转到另一个页面，或在同一文档的不同位置间导航。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin: 10px 0; color: #0d47a1;">
  <strong>最佳实践：链接文本</strong><br>
  为了 SEO 和无障碍访问，链接文本应当具有描述性（例如“查看产品列表”），严禁使用“点击这里”这种无意义的词汇。
</div>

## 超链接标签

HTML 使用 `<a>` 标签（Anchor，锚）来定义超链接。

- **行内元素**：`<a>` 是行内元素（Inline），可以包裹文本或图片。
- **基本交互**：默认情况下，未访问的链接通常显示为带有下划线的蓝色文本，点击后会改变颜色。

## 核心属性详解

### href (Hypertext Reference)
指定链接目标的 URL，是超链接必须具备的属性。
- **绝对 URL**：指向完整的网址（如 `https://www.google.com`）。
- **相对路径**：指向当前站点内的文件（如 `/about.html`）。
- **占位符**：若暂时没有确定目标，可使用 `href="#"` 或 `href="javascript:void(0);"`。

### target
指定链接打开的上下文方式。

## 功能分类与演示

### 1. 外部链接与内部链接
最基础的跳转功能。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- 源码 --&gt;<br>
    &lt;!-- 外部链接：跳转到其他网站 --&gt;<br>
    &lt;a href="https://www.example.com" target="_blank" rel="noopener noreferrer"&gt;访问示例网站 (新窗口)&lt;/a&gt;<br>
    <br>
    &lt;!-- 内部链接：跳转到当前目录下的文件 --&gt;<br>
    &lt;a href="index.html"&gt;返回首页&lt;/a&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <a href="https://www.example.com" target="_blank" rel="noopener noreferrer" style="color: #2196f3; text-decoration: underline; margin-right: 20px;">访问示例网站 (新窗口)</a>
    <a href="#" style="color: #2196f3; text-decoration: underline;">返回首页</a>
  </div>
</div>

### 2. 锚点链接 (Page Anchor)
用于在长页面中快速跳转到指定位置。实现原理是 `href` 属性的值对应目标元素的 `id` 属性。

<div style="border: 1px solid #ddd; border-radius: 4px; overflow: hidden; margin: 20px 0;">
  <div style="background-color: #f5f5f5; padding: 10px; border-bottom: 1px solid #ddd; font-family: Consolas, monospace; font-size: 14px; color: #333;">
    &lt;!-- 源码 --&gt;<br>
    &lt;!-- 点击链接跳转到 id 为 section1 的位置 --&gt;<br>
    &lt;a href="#section1"&gt;跳转到第一章&lt;/a&gt;<br>
    <br>
    &lt;div style="height: 50px;"&gt;...模拟中间内容...&lt;/div&gt;<br>
    <br>
    &lt;!-- 目标位置 --&gt;<br>
    &lt;h3 id="section1"&gt;第一章内容&lt;/h3&gt;
  </div>
  <div style="padding: 20px; background-color: #fff;">
    <a href="#section1_demo" style="color: #2196f3; text-decoration: underline;">跳转到第一章</a>
    <div style="height: 30px; border-left: 2px dashed #ccc; margin: 10px 0 10px 10px;"></div>
    <h3 id="section1_demo" style="margin: 0;">第一章内容</h3>
  </div>
</div>

### 3. 伪协议链接
`href` 不仅可以指向网页，还可以触发系统级应用。

- **邮件**：`mailto:support@example.com`
- **电话**：`tel:+8613800138000` (在移动端常用)
- **短信**：`sms:10086`
